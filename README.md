# 🔔 기획 배경 및 소개

도저히 혼자 벌레를 잡을 자신이 없을 때, 벌레를 잘 잡는 동네 사람들에게 도움을 구할 수 있는 웹앱 서비스입니다.
<br>

# 💼 폴더 구조
```bash
─ src <br>
   ├── __mock__ (for useSelector, useDispatch, API mocking) <br>
   ├── __test__ (for pages)
   ├── components
   │    ├── layout
   │    └── utils
   ├── core
   │    ├── api
   │    │   └── instance
   │    └── redux
   │        ├── module
   │        └── store
   ├── hooks
   ├── pages
   │    ├── auth
   │    └── board
   └── styles
```
